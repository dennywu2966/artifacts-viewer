# artifacts-viewer

Central repository for storing and viewing artifacts (documents, presentations, markdown, images, diagrams).

## Directory Structure

```
artifacts-viewer/
├── docs/       # PDF, DOC, DOCX files
├── slides/     # PPT, PPTX files
├── markdown/   # MD, MDX files
├── images/     # PNG, JPG, JPEG, GIF
├── diagrams/   # SVG files
└── other/      # Other file types
```

## Usage

Artifacts are automatically organized by category based on file extension.
